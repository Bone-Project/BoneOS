###Command `reboot` manual

**NAME**

> `reboot` - reboots/restarts the system.

**SYNOPSIS**

> `reboot [option]`
    
**DESCRIPTION**

> `reboot` allows a system administrator to reboot/restart the system using the command line.

**OPTIONS**

>- `--help`: prints the help for this command in the terminal

**ALSO SEE**

>- `poweroff`
