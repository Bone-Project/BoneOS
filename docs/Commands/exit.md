###Command `exit` manual

**NAME**

> `exit` - exits the current shell instance

**SYNOPSIS**

> `exit [option]`
    
**DESCRIPTION**

> `exit` exits the current shell instance

**OPTIONS**

>- `--help`: prints the help for this command in the terminal

**SHORTCUTS**

> <kbd>Control</kbd> + <kbd>D</kbd> is a shortcut for this command.

**ALSO SEE**

>- `boneshell`
