## Compiling OS

###Linux
Clone this GIT Repo
```sh
git clone --recursive https://github.com/amanuel2/BoneOS.git
```

Install GRUB
```sh
sudo apt install grub
```

Install Qemu
```sh
sudo apt-get install qemu
```

and then you can run
```sh
make kernel_multiboot
```
or
```sh
make cd_rom
```
